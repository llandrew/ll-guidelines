Naming / Namespacing

Extensions:

Good: .html, .php, .scss, .css, .js

Bad: .htm, .php5, .sass

Editor Settings

Use soft tabs with two spaces (the only way to guarantee the code renders the same across all environments).

.editorconfig file settings

indent_style = space
indent_size = 2
tab_width = default
end_of_line = lf
charset = utf-8
trim_trailing_whitespace = true
insert_final_newline = true

Sublime custom settings

draw_white_space = all
detect_indentation = true
trim_trailing_white_space_on_save = true
