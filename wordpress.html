Using Plugins

Functions and Template Tags

Because plugins can be deactivated at any time and that action must not impact the function of the theme, we should always be adding sanity checks to all plugin usage.

Bad...


/**
 * Call Template Tag
 */

my_template_tag();

Better


/**
 * Call Template Tag
 */

if ( function_exists( 'my_template_tag' ) )
  my_template_tag();
